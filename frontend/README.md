# SpotOnSight Frontend

Vue 3 application for the SpotOnSight product surface.

## Commands

```bash
npm install
npm run dev
npm run build
npm run test:run
```

## Mobile Build Bridge

- Active Capacitor CLI config: `frontend/capacitor.config.json`
- Native shells: `mobile/capacitor/android/` and `mobile/capacitor/ios/`

## Key Directories

- `src/components/`
- `src/views/`
- `src/router/`
- `src/stores/`
- `src/services/`
